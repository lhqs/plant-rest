## 物联完管理系统后台接口服务

* 种植系统后台接口服务，提供了移动端或者前端获取数据的接口等常用功能，便于系统维护，优化，可扩展；
* 系统所涉及的技术包括：SpringMVC + Spring + MyBatis + MySql + RESTful + Tomcat + 七牛云对象存储；
